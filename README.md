# An Unhinged Fitness Workbook — 2026 (Preview)

Interactive web workbook (HTML/CSS/JS). No backend. Saves in your browser (localStorage).

## Use locally
Open `index.html` in a browser.

## Hosting on GitHub Pages
1. Create a new repo on GitHub (public is easiest for the preview).
2. Upload these files.
3. GitHub → Settings → Pages:
   - Source: Deploy from a branch
   - Branch: main / Folder: / (root)
4. Your site will be live at https://YOURNAME.github.io/REPO/

## iPhone Calendar export
Use Calendar → Export .ics.
On iPhone: open the downloaded .ics file (Files app, AirDrop, Mail) → “Add to Calendar”.

## Customize CTA
In index.html, replace the https://www.etsy.com/ link with your Etsy listing link.
